# post-covid-care

## Instruction
 - Download this package in zip, open this project in xCode. Update your team and identifier info in > Signing&Capabilities (Make sure you already have a development account.)
 - By default, you are using this app with a demo API provided by HCL. If you have a paid API access, you can change the default APIkey to yours in [AppSettings.swift](AppSettings.swift)
